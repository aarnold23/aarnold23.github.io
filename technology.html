Technology Stack

[Circuit Board Icon]

Raspberry Pi 4

Central computing unit with 4GB RAM, processing video feeds and controlling LEDs via GPIO pins 17-22.

[Camera Icon]

Picamera2 Module

Captures real-time video at 640x480 resolution for vehicle detection in two lanes.

[CPU Icon]

YOLOv5-nano

Lightweight model with 1.9M parameters, optimized for real-time detection on edge devices, achieving 85% accuracy.

[Code Icon]

Python Framework

Integrates OpenCV, YOLOv5, and RPi.GPIO for video processing, detection, and LED control.
