Project Results

Key Findings

  • [Check Circle Icon] Achieved 85% detection accuracy for toy cars using YOLOv5-nano.
  • [Check Circle Icon] Reduced wait times by 20% compared to fixed-timing lights.
  • [Check Circle Icon] Dynamic timing (10s for 3 cars, 15s for 5 cars) responded to traffic patterns.
  • [Check Circle Icon] Stable performance over 8-hour test periods.

Future Improvements

  • Reduce camera latency by optimizing YOLOv5-nano to 15 FPS.
  • Automate lane-specific counting with edge detection.
  • Integrate pedestrian detection for crosswalk compatibility.
  • Develop a dashboard for real-time traffic monitoring.
