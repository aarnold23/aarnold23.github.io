How It Works

System Components

Hardware: Raspberry Pi 4, Picamera2 module, six LEDs (red, yellow, green for two lanes) connected via a breadboard.

Software: Python with OpenCV, YOLOv5-nano for vehicle detection, and RPi.GPIO for LED control.

Workflow

The Picamera2 captures video of toy cars in two lanes. YOLOv5-nano processes frames to count cars with 85% accuracy. Based on counts, the system adjusts green light durations (e.g., 10s for 3 cars in Lane A, 15s for 5 cars in Lane B). Data is logged for analysis.

Kampala Application

Imagine this system at Kampala Road, detecting matatus and bodas in real time to optimize traffic flow at busy intersections like City Square.
