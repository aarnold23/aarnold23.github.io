Sample Code


import RPi.GPIO as GPIO
GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)  # Red LED for Lane A
GPIO.output(17, GPIO.HIGH)  # Turn on red light

Running the Code

Clone the repository to a Raspberry Pi 4, install dependencies (OpenCV, YOLOv5, RPi.GPIO), and run main.py with a connected Picamera2 and LEDs.
