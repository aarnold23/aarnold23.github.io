Key Files

  • main.py: Orchestrates YOLOv5 detection, LED control, and logging.
  • log.csv: Stores data, e.g., 2025-07-18 10:00, Lane A: 3 cars, Lane B: 5 cars, Lane A Green: 10s.
  • config.yaml: YOLOv5-nano configuration with 0.3 confidence threshold.

Sample Code: LED Control


import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)  # Red LED Lane A
GPIO.setup(18, GPIO.OUT)  # Green LED Lane A

def set_traffic_light(lane, state, duration):
    if lane == 'A' and state == 'green':
        GPIO.output(17, GPIO.LOW)
        GPIO.output(18, GPIO.HIGH)
        time.sleep(duration)

Sample Code: YOLOv5 Detection


import cv2
from yolov5 import YOLOv5

model = YOLOv5('yolov5n.pt', device='cpu')
cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()
    results = model.predict(frame, conf_thres=0.3)
    car_count = len(results.xyxy[0])
    print(f"Cars detected: {car_count}")

Setup Instructions

  1. Clone: git clone https://github.com/Komukamatracy/AUTOMATED-TRAFFIC-LIGHT-MANAGEMENT-SYSTEM-PROJECT
  2. Install: pip install opencv-python torch yolov5 RPi.GPIO
  3. Connect Picamera2 and LEDs (GPIO 17-22).
  4. Run main.py.
